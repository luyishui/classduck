# 任务33：侧边栏交互动效与节数语义及待办新建细化

## 1. 本轮目标
1. 修复课程表侧边栏动画质感、分段节数语义、控件配色与滚轮问题。
2. 修复待办侧边栏垃圾箱位置与文本对齐。
3. 修复新建待办课程联动、截止日期/时间显示与任务类型“更多”交互。
4. 强化我的页面统计刷新时机。

## 2. 本轮主要改动
- `classduck_app/lib/features/schedule/ui/schedule_page.dart`
- `classduck_app/lib/features/todo/ui/todo_page.dart`
- `classduck_app/lib/features/todo/ui/new_todo_page.dart`
- `classduck_app/lib/features/profile/ui/profile_page.dart`

## 3. 关键点
### 3.1 课程表侧边栏
1. 动效升级：抽屉开启改为 `fade + slide + scale` 复合动画，关闭同曲线回退。
2. 课表设置：课表名称行与“新建课表”输入行统一左对齐。
3. 分段语义改正：
   - 上午课程：`上午课程节数`
   - 下午课程：`下午课程节数`（承接上午后续节次）
   - 晚上课程：`晚上课程节数`（承接下午后续节次）
4. 加减按钮颜色按分组主题深色化，不再全蓝色。
5. 每周起始日滚轮开启循环滚动（可上滑到前一项），并修复选中遮罩遮字问题。
6. 开学日期滚轮仅显示阿拉伯数字（去掉 年/月/日 后缀）。

### 3.2 待办侧边栏
1. 垃圾箱移动到右侧。
2. 任务类型文字改为左对齐。
3. 删除行为保持“删除类型并联动删除该类型下全部待办”。

### 3.3 新建待办
1. 课程选项由课表真实课程动态加载（有课程才显示关联课程）。
2. 截止日期与截止时间分块显示标签，保持旋轴选择。
3. 任务类型默认最多显示 3 个，超过后显示 `...` 入口，点击弹出完整选择面板。
4. 调整滚轮选中遮罩，避免覆盖文本可读性。

### 3.4 我的页面
1. 打开“已上课程/已完成待办”前先刷新统计。
2. 从设置页返回后自动刷新统计，避免数字滞后。
