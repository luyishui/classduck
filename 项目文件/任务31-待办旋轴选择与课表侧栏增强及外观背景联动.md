# 任务31：待办旋轴选择与课表侧栏增强及外观背景联动

## 1. 目标
1. 按反馈将“新建待办”的日期与时间改为旋轴（滚轮）选择。
2. 强化课表侧边栏为可滚动的完整抽屉，补齐快捷操作入口。
3. 完成“外观与主题”到课表页面背景的真实联动（上传后即时生效）。

## 2. 变更文件
- `classduck_app/lib/features/todo/ui/new_todo_page.dart`
- `classduck_app/lib/features/schedule/ui/schedule_page.dart`
- `classduck_app/lib/features/settings/ui/appearance_page.dart`
- `classduck_app/lib/features/settings/data/appearance_state.dart`

## 3. 关键实现
### 3.1 新建待办旋轴选择
1. 日期与时间入口改为底部弹出的滚轮选择器。
2. 使用 `CupertinoDatePicker`，顶部包含“取消/确定”操作。
3. 日期仅覆盖年月日，时间仅覆盖时分，避免相互污染。

### 3.2 课表侧边栏增强
1. 顶部改为“鸭 + 上课鸭”居中品牌头。
2. 课表列表保持可滚动长抽屉结构。
3. 新增快捷操作卡片：手动添加课程、教务导入课程。

### 3.3 外观与课表背景联动
1. 新增全局外观状态仓储 `AppearanceStore`。
2. 外观页上传图片后写入内存字节并记录文件名。
3. 课表页监听外观状态，存在背景图时用 `DecorationImage` 覆盖默认背景。
4. 清除背景图片后课表页立即恢复默认背景。
