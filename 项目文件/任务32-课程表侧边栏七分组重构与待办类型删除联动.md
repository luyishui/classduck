# 任务32：课程表侧边栏七分组重构与待办类型删除联动

## 1. 目标
1. 按《课程表侧边栏 操作逻辑说明与PRD》重构侧边栏为 7 分组抽屉结构。
2. 对齐上课鸭.pen 的课程表侧边栏视觉与交互（遮罩、滑入、分组卡片色系、展开内容区）。
3. 在待办侧边栏给每个待办类型增加左侧黄色垃圾箱，点击后删除类型并联动删除该类型下全部待办。
4. 保证“类型删除”会同步影响待办主页分组和新建待办类型列表。

## 2. 变更文件
- `classduck_app/lib/features/schedule/ui/schedule_page.dart`
- `classduck_app/lib/features/schedule/data/schedule_repository.dart`
- `classduck_app/lib/features/todo/ui/todo_page.dart`
- `classduck_app/lib/features/todo/data/todo_repository.dart`

## 3. 关键实现
### 3.1 课程表侧边栏
1. 侧边栏改为 7 个分组：课表设置、上课时间、上午课程、下午课程、晚上课程、周数设置、开学日期。
2. 仅一个分组默认展开（课表设置），其余折叠，可点击切换展开状态。
3. 顶层增加遮罩+背景模糊，点击遮罩或关闭按钮可关闭抽屉。
4. 课表设置支持新建课表（输入框 + 勾叉确认）与多课表切换（当前课表勾选）。
5. 上课时间支持课时/课间步进和“一键批量对齐”，同步更新左侧时间轴。
6. 周数设置支持学习周数步进与每周起始日滚轮；开学日期支持年/月/日滚轮并回显。

### 3.2 侧边栏配置联动
1. 新增 `_ScheduleConfig` 运行态模型（课时模板、节次分段、周设置、开学日期）。
2. 按 `tableId` 维护配置映射，切换课表时同步切换配置。
3. 配置写入 `course_table.class_time_list_json` 与 `semester_start_monday`，保证课表间配置隔离。

### 3.3 待办类型删除联动
1. 待办侧边栏每个类型左侧常驻黄色垃圾箱图标。
2. 点击垃圾箱弹确认框，确认后删除该类型。
3. 删除类型时同步调用仓储删除该 `task_type` 下全部待办。
4. 删除后同步更新 `_typeOrder/_typeLabels`，新建待办页收到的新类型集合自动收敛。
