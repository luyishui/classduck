# 任务26-移动端对齐修复（加号菜单_分栏尺寸_新建待办_我的弹窗）

## 1. 任务目标
根据用户本轮反馈，修复当前与 PRD/Pen 偏差较大的移动端交互与页面实现，重点覆盖：
1. 课程表加号菜单必须是列表型浮层，而不是底部抽屉。
2. 选择学校与待办分栏的“激活高亮范围”不足问题。
3. 新建待办页面未按 PRD 落地的问题。
4. 我的页“已上课程/已完成待办弹窗”“今日状态弹窗”从占位改为可用实现。

## 2. 本次改动文件
1. classduck_app/lib/features/schedule/ui/schedule_page.dart
- 将加号菜单从 `showModalBottomSheet` 改为右下角悬浮列表浮层。
- 菜单三项采用 136x48 的独立白色胶囊项，新增黄色圆形关闭按钮。

2. classduck_app/lib/features/import/ui/import_school_list_page.dart
- 移除默认 TabBar 方案，改为 Pen 风格的宽胶囊分栏（80 宽项）。
- 支持左右滑动手势切换分栏激活态，确保黄色背景覆盖的是“标签范围”而非仅文字区域。

3. classduck_app/lib/features/todo/ui/todo_page.dart
- 待办顶部分栏改为更贴近 Pen 的 44 高容器与整段高亮覆盖。
- 侧边栏按钮由占位弹窗升级为“任务类型管理”弹窗，支持新增自定义任务类型并同步主列表分组。

4. classduck_app/lib/features/todo/ui/new_todo_page.dart
- 从英文通用表单重构为中文 PRD 页面：
  - 标题栏（关闭 + 居中标题）
  - 计划输入大卡片
  - 任务类型胶囊
  - 关联课程胶囊（含“更多”自定义输入）
  - 截止日期与时间选择
  - 底部 `+ 添加待办` 主按钮
- 提交校验改为中文文案，且要求计划与关联课程。

5. classduck_app/lib/features/profile/ui/profile_page.dart
- “已上课程”弹窗：改为可搜索 + 分组抽屉只读查看（按课表分组）。
- “已完成待办”弹窗：改为可搜索 + 分组抽屉只读查看（按作业/考试/竞赛分组）。
- “今日状态”弹窗：改为搜索、选择、新建心情（带输入区与确认/取消）并回写主页状态。

6. classduck_app/lib/features/settings/ui/about_page.dart
- 修复 lint：去除不必要的 `this.` 限定符。

## 3. 核心函数与关键实现
### 3.1 SchedulePage
- `_openAddMenu()`：
  - 改为 `showGeneralDialog`。
  - 列表菜单固定在右下角，含 `手动添加/教务添加/拍照添加`。
  - 点击项后返回 action 并执行原跳转/调用逻辑。

### 3.2 ImportSchoolListPage
- 分栏改为 `_SchoolLevelTab` 组件。
- 顶部分栏容器支持 `onHorizontalDragEnd`，实现左滑右滑切换激活分栏。

### 3.3 TodoPage
- `_openSidebarPlaceholder()` 已替换为可操作逻辑：
  - 展示当前任务类型。
  - 支持新增类型并实时写入 `_typeOrder/_typeLabels`。
- `_typeOrder` 改为动态列表，不再是常量。

### 3.4 NewTodoPage
- 重构页面结构并新增组件：
  - `_SelectPill`（任务类型/课程胶囊）
  - `_DateTimeCard`（日期时间行）
- 关键交互：`_pickDate()`、`_pickTime()`、`_inputCustomCourse()`、`_submit()`。

### 3.5 ProfilePage
- 新增 `_showReadonlyGroupedModal()` + `_ReadonlyGroupedModal`：
  - 可搜索、可展开分组、只读内容列表。
- 新增 `_buildDoneCourseGroups()`：
  - 从课表与课程数据中按“已上”规则计算并分组。
- 新增 `_MoodOptionTile` 与增强 `_openMoodModal()`：
  - 支持选择已有心情、搜索过滤、新建心情并回写。
