# 任务40：导入规则内置化与三分栏收敛

## 1. 本轮目标
1. 导入分栏从“专科/本科/硕士/通用”收敛为“本/专科/硕士/通用”。
2. 通用分栏仅显示教务系统入口，不直接显示学校列表。
3. 按清单将规则内置到本地配置（先不接云端自动热更新）。
4. 保留强制更新与规则热更新接口说明。

## 2. 代码落地
### 2.1 前端分栏调整
- 文件：`classduck_app/lib/features/import/ui/import_school_list_page.dart`
- 变更：
1. 分栏改为 3 个：本/专科、硕士、通用。
2. 默认激活分栏改为本/专科。
3. 过滤键改为：`undergraduate/master/general`。
4. 学校卡片移除 `>` 箭头与 `delay` 文案，保留纯学校名称点击导入。
5. 下载/问号按钮改为右下角黄色圆按钮布局（问号位于下载按钮下方）。
6. 中文学校名分组改为拼音首字母（A-Z 导航可用于本/专科与硕士分栏）。

### 2.1.1 导入执行页登录链路
- 文件：`classduck_app/lib/features/import/ui/import_execution_page.dart`
- 变更：
1. 增加“登录页URL”输入框（可编辑）。
2. 增加“打开登录页”按钮，用户先登录教务系统。
3. 下载按钮文案改为“下载并导入课表”。
4. 说明文案明确当前 parser 仍为占位实现。
5. 进入页面后自动尝试打开当前登录URL。
6. 下载/问号改为右下角常驻黄色圆按钮，保持与主流程操作位一致。

### 2.2 级别归一
- 文件：`classduck_app/lib/features/import/domain/school_config.dart`
- 变更：
1. `junior` 统一映射为 `undergraduate`。
2. 含“学校/学院/大学”等关键词默认归并到本/专科。

### 2.2.1 学校配置离线兜底
- 文件：`classduck_app/lib/features/import/data/school_config_repository.dart`
- 变更：
1. 先请求后端 `/v1/config/schools`。
2. 请求失败自动读取应用内置 `assets/config/schools.builtin.json`。
3. 解决未启动后端时“Failed to fetch”问题。
4. 内置文件与后端样例保持同步，便于团队在 GitHub 环境直接调试。

### 2.3 本地内置学校配置
- 文件：`backend/src/config/schools.sample.json`
- 变更：
1. 新版内置数据共 81 条（含教务系统入口 + 学校项）。
2. 通用分栏入口含 URP、树维、青果、金智、正方等系统项。
3. 学校项按本/专科或硕士归类。
4. 修正：德阳通用电子科技学校归并到本/专科。
5. `initialUrl/targetUrl` 当前使用占位地址，后续按各系统真实登录页逐步替换。
6. 新增硕士测试项：`西安交通大学`。

### 2.4 适配规则预置
- 文件：`backend/src/config/adapter-rules.sample.json`
- 变更：
1. 通用 URP 规则加宽 host/path 匹配提示。
2. 脚本地址用 `local://` 协议占位，表示本地内置解析入口。
